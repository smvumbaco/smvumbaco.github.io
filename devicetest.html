Emerald Innovations: Device Evaluation Program

While I was at Emerald, the biggest initiative on the hardware team was transitioning from an older hardware model to a newer model. This newer version of the health monitoring device outperformed the older model in performance, size, weight, and functionality. While the device performed well, there was no infrastructure for properly testing these new devices to evalute their functionaity before they were shipped to participants. To address this deficiency, I developed a testing regimen that evaluated every antenna, sensor, switch, and communication bus in the device assembly. This program adapted the existing program used in the older hardware while adding dozens of scripts to account for the new features introduced in the newer hardware. By the time I left Emerald, this software library was in use on an almost daily basis to quickly and accurately determine if new devices were 100% ready for export.
